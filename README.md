# My AI Journey

> Public blog about building AI agents, automating workflows, and learning in public.
> Built with [Hugo](https://gohugo.io) + [PaperMod](https://github.com/adityatelange/hugo-PaperMod).

## Setup

```bash
git clone git@github.com:Ejaramillo1/My-AI-Journey.git
cd My-AI-Journey
hugo mod get
hugo server
```

## Deploy

Push to `main` → GitHub Actions builds and deploys to GitHub Pages.
